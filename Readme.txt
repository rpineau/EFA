This plugin allows you to control a PaleWave EFA focuser controller ( http://planewave.com/products-page/general-accessories/efa-kit-electronic-focuser/ ) with TheSkyX Professional.

Pre-compiled versions for OS X, Windows, Unbuntu X86_64 and Rapsberry Pi Pixel can be found there :
http://www.rti-zone.org/macosx_x2focuser_plugins.php
